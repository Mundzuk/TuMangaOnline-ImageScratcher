# TuMangaOnline-Scratcher
Saca imagenes de TuMangaOnline para descargar "Manhua/Manhwa/Manga/One-shot" en tu escritorio local. Solo debes insertar el link del capitulo ejemplo: "https://lectortmo.com/viewer/258d85261eb34e50e4dc22c03a210337/paginated". Tambien funciona en las redirecciones de TMO,  como por ejemplo "animalcanine", "otakuworldgames", "wtechnews", etc.
PD: al terminar la descarga del capitulo automaticamente empezara a descargar el siguiente.
